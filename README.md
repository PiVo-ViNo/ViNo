# [Test] Usage

1. Compiler for Visual Novel Script Language (.vnsf):

   `vino-compiler.exe -o <my_vn_directory> <my_vn_script>.vnsf`

2. `Vinovima.exe`, `m_vm_inst.bin` and `res` must be in the root of `./vm` directory
